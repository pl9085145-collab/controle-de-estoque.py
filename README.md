# Sistema de cotrole de estoque 
Este projeto é um sistema de linha de comando (CLI) desenvoido em python para gerenciar o estoque de uma loja de eletrônicos. 
## Funcionalidades
-**Cadastro:** Adiciona novos produtos com preços e quantidade.
-**Consulta:**Lista todos os itens do estoque.
-**Edição:**Permite atualizar valores e quantidades. 
-**Exclusão:**Remover itens do inventário.
##Como usar
Basta executar o arquivo 'estoque.py'em um ambiente Python e seguir as instruções do menu numéricos.
